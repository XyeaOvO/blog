*   **视角一：列组合**
    *   `C` 的 **列** 是 `A` 的 **列** 的线性组合。
    *   权重来自 `B` 的对应 **列**。
    *   几何意义：将 `A` 所代表的变换，作用于 `B` 的每一个列向量上。

*   **视角二：行组合**
    *   `C` 的 **行** 是 `B` 的 **行** 的线性组合。
    *   权重来自 `A` 的对应 **行**。
    *   意义：将 `B` 所代表的数据（或变换），通过 `A` 中每一行的权重进行重新组合

```cpp
[ a  c ] [ e  g ]   =   [ a*e + c*f   a*g + c*h ]  <-- C 的第一行
[ b  d ] [ f  h ]       [ b*e + d*f   b*g + d*h ]  <-- C 的第二行
   (A)      (B)                 (C)
```
