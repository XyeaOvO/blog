### 第三阶段：完美配置（防坑指南）

这是“最佳实践”的核心，解决“找不到库”和“闪退”问题。

#### 1. 解决“No Libraries Found” (找不到库)
这是最常见的问题。Proteus 8把库文件从安装目录分离到了 `ProgramData` 目录。
*   **方法**：
    *   找到桌面的 Proteus 8图标，右键 -> **属性** -> **兼容性**。
    *   勾选 **“以管理员身份运行此程序”**。
    *   （可选）如果还是报错，勾选“以兼容模式运行”，选择 Windows 7。
    *   *解释：Proteus需要权限去读取C盘系统目录下的库文件，不给管理员权限经常读不到。*

#### 2. 配置第三方库（Arduino/STM32/传感器）
自带的库没有Arduino，学生做课设必装。
*   **下载库**：去网上下载 `Arduino Library for Proteus`（包含 `.LIB` 和 `.IDX` 文件）。
*   **放置位置（重点）**：
    *   不要放在安装目录的 `Program Files` 里！
    *   **正确路径**：`C:\ProgramData\Labcenter Electronics\Proteus 8 Professional\LIBRARY`
    *   *提示：找不到 `ProgramData`？按 `Win+R`，输入 `%ProgramData%` 回车即可直达。*

#### 3. 设置自动保存（防崩溃）
Proteus仿真复杂电路时偶尔会崩，学生党甚至可能因此挂科。
*   **设置**：打开 Proteus -> 菜单栏 `System` -> `System Settings`。
*   **修改**：
    *   勾选 **"Auto-Save"**。
    *   时间间隔设为 **5~10分钟**。
    *   Backup数量设为 **3** 份。

### 第四阶段：学生党操作最佳实践

为了让你像个老手一样操作，建议养成以下习惯：

#### 1. 界面优化（网格与对齐）
很多新手画图线歪歪扭扭，因为没设置好网格。
*   **开启网格**：按快捷键 `G` 可以在“无网格”、“点网格”、“线网格”之间切换。建议画图时使用**点网格**。
*   **吸附设置**：菜单栏 `View` -> `Snap 10th` (0.1英寸)，这是画原理图的标准间距，引脚最容易对齐。

#### 2. 必须记住的快捷键
别用鼠标一个个点菜单了，效率极低。
*   **F8**：一键居中显示整个电路（图找不到了就按它）。
*   **P**：Pick，打开元件库搜索元件。
*   **双击右键**：直接删除元件（慎用，很爽但容易误删）。
*   **R**：在放置元件时，按 `+` 或 `-` 号旋转（有些版本是按 `R` 键）。
*   **Ctrl + Z**：撤销（救命键）。

#### 3. 常见仿真报错速查
*   **"VCC not found"**：你可能把电源脚隐藏了。
    *   *解决*：菜单 `Design` -> `Configure Power Rails` -> 把 `VCC/VDD` 添加到右边的 `VCC` 网络里。
*   **"Real Time Simulation failed" (CPU占用过高)**：
    *   *解决*：双击你的单片机或高频元件，把 `Clock Frequency` 改低一点（比如从12MHz改成1MHz，只影响仿真速度，不影响逻辑）。
