# 第一部分：核心考点精讲
> **使用说明**：本部分涵盖了PPT中所有的定义、数据、机制和分类。加粗部分为高频考点。

## A. 神经系统的细胞 (Cells of the Nervous System)

### 1. 神经系统概览
*   **中枢神经系统 (CNS)**
    *   **脑 (Brain)**
    *   **脊髓 (Spinal cord)**
*   **周围神经系统 (PNS)**
    *   **脑神经**：12对 (Cranial nerves)
    *   **脊髓神经**：31对 (Spinal nerves)
    *   内脏神经 (Visceral nerves)

### 2. 大脑关键数据 (Brain Facts)
*   **重量**：成人脑重 1200~1500g。
*   **占比**：占体重的 2%~2.5%。
*   **能量消耗**：极高，消耗人体 **20%** 的总能量。
*   **细胞数量**：
    *   神经元 (Neurons)：约 $10^{13}$ 个。
    *   突触 (Synapses)：约是神经元数量的 1000~4000 倍。
    *   胶质细胞 (Glia)：约是神经元数量的 10~50 倍。

### 3. 爱因斯坦大脑的特异性 (Case Study)
*   *考试意义：用于分析智力与大脑结构的非绝对关系。*
*   重量：只有 1230克（轻于常人平均值）。
*   **形态差异**：中央沟（中央裂）位置形状异常。
*   **细胞比例**：左顶叶神经元与神经胶质细胞比例异于常人。
*   **区域发达**：左右半球**顶叶下区域**比普通人宽出 15%，且更厚（该区域与数学、空间推理有关）。

### 4. 神经元 (Neuron) —— 结构与功能单位
#### 4.1 神经元学说 (Neuron Doctrine)
*   **提出者**：Santiago Ramón y Cajal (1852–1934)。
*   **核心观点**：神经元是神经系统的结构单位；细胞在解剖上是**分离**的（非相互贯通），通过特定结构（突触）接触。
*   **基本特性**：
    *   数量占CNS的 10%。
    *   **可兴奋**细胞，代谢率高。
    *   完全分化，**不再分裂、增生**（意味着受损难再生）。

#### 4.2 神经元的四个功能部位
1.  **胞体和树突**：接受和整合信息的部位。
2.  **轴突始端 (轴丘 Axon hillock)**：**产生动作电位**的起始部位（Key Point）。
3.  **轴突**：传导神经冲动的部位。
4.  **轴突末梢**：递质释放的部位。

#### 4.3 树突 (Dendrites) vs. 轴突 (Axon) 【必考对比】

| 特征 | 轴突 (Axon) | 树突 (Dendrites) |
| :--- | :--- | :--- |
| **数量/长度** | 每个神经元**只有一根**，长且分支少 | 多且可变，短，有多级分支 |
| **形态** | 细长均匀，表面光滑 | 呈树枝状，粗糙 |
| **起始阶段** | 远离胞体（轴丘发出） | 胞体的延伸 |
| **末端** | 不逐渐变细 | 逐渐变细 |
| **棘刺 (Spines)** | **无** | **常有树突棘** |
| **髓鞘** | 部分有髓鞘化 | 极少有髓鞘 |
| **功能** | **输出端** (Output)，传出信号 | **输入端** (Input)，接受、整合信息 |
| **突触位置** | 通常为突触前结构 | 通常为突触后结构 |
| **蛋白合成** | 基本无（无核糖体/粗面内质网） | 可局部合成 |

#### 4.4 树突棘 (Dendritic Spine)
*   **定义**：树突表面细颈连接的小球状突起，是形成突触的主要部位。
*   **功能**：兴奋传递点；参与**学习记忆**；具有**可塑性**。
*   **动态变化**：
    *   随发育先增后减（修剪 Pruning）。
    *   **病理变化**：在阿尔茨海默病 (AD) 中，树突棘数量明显减少，变得细长。

#### 4.5 神经元的分类
1.  **按突起数量**：
    *   **单极/假单极**：T型分支，常见于脊髓背根神经节（感觉神经元）。
    *   **双极**：视网膜双极细胞。
    *   **多极**：大多数神经元（如脊髓运动神经元、海马锥体细胞）。
2.  **按反射弧位置**：
    *   传入（感觉）：外周 -> 中枢。
    *   中间（联络/整合）：起联络作用。
    *   传出（运动）：中枢 -> 效应器。
3.  **按突触后影响**：兴奋性 vs. 抑制性。
4.  **按释放递质**：胆碱能、肾上腺素能、氨基酸能、胺能、肽能。
5.  **按轴突长短**：
    *   Golgi type I：长轴突（投射神经元）。
    *   Golgi type II：短轴突（局部回路神经元）。

### 5. 神经胶质细胞 (Glial Cells)
*   **特点**：数量多 (10-50倍于神经元)、体积小、无轴突树突之分、**可分裂增殖**、**不能传导动作电位**。
*   **分类与功能表 【重点背诵】**

| 所在系统 | 细胞名称 | 英文名 | 核心功能 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **CNS (中枢)** | **星形胶质细胞** | Astrocyte | 1. 形成**血脑屏障 (BBB)**<br>2. 支持/分隔神经元<br>3. 调节离子平衡 (吸收K+)<br>4. 参与代谢 | 脑内数量最多，体积最大 |
| **CNS (中枢)** | **少突胶质细胞** | Oligodendrocyte | 形成**中枢**神经纤维的**髓鞘** | 增加传导速度 |
| **CNS (中枢)** | **小胶质细胞** | Microglia | **免疫细胞**，吞噬作用，“清道夫” | 最小，清除死亡细胞/病原体 |
| **PNS (外周)** | **施万细胞** | Schwann cell | 形成**外周**神经纤维的**髓鞘** | 胞内富含脂质，绝缘 |
| **PNS (外周)** | **卫星细胞** | Satellite cell | 包裹神经节内胞体，保护作用 | 位于神经节 |

*   **临床意义**：
    *   **血脑屏障 (BBB)**：星形胶质细胞参与维持。BBB破坏是AD病的早期标志。
    *   **癫痫 (Epilepsy)**：星形胶质细胞功能紊乱 -> 细胞外 K+ 浓度升高 -> 神经元兴奋性异常增高 -> 放电。

## B. 神经电信号 (Neural Signals)

### 1. 神经元膜基础
*   **脂质双分子层**：疏水尾相对，亲水头朝外，阻隔水溶性物质（绝缘）。
*   **膜蛋白**：
    *   **离子通道**：具有离子选择性、门控特性（电压门控/化学门控）。
    *   **钠钾泵 (Na+-K+ Pump)**：**主动运输**，消耗ATP（占脑能耗70%）。
        *   **机制**：每消耗1个ATP，**泵出 3个 Na+，泵入 2个 K+**。
        *   **结果**：维持膜内高 K+，膜外高 Na+。

### 2. 静息电位 (Resting Potential, RP)
*   **定义**：细胞未受刺激时，膜两侧的电位差（内负外正）。
*   **数值**：约 **-65mV** (-60 ~ -90mV)。
*   **产生机制**（两个条件）：
    1.  **离子分布不均**：膜外 Na+/Cl- 高，膜内 K+ 高。
    2.  **膜通透性差异**：静息时，膜对 **K+** 通透性大（K+漏通道开放），对 Na+ 通透性极小。
*   **过程**：K+ 顺浓度梯度外流 -> 膜内负电荷增加 -> 电场力阻止 K+ 继续外流 -> 达到**电化学平衡**。

### 3. 动作电位 (Action Potential, AP)
*   **定义**：阈上刺激引起膜电位快速去极化和复极化，可远传。
*   **产生机制（离子流）**：
    1.  **去极化 (上升相)**：刺激达到阈值 -> 电压门控 **Na+ 通道开放** -> **Na+ 快速内流** -> 膜电位由负变正。
    2.  **复极化 (下降相)**：Na+ 通道失活关闭，电压门控 **K+ 通道开放** -> **K+ 快速外流** -> 膜电位恢复负值。
    3.  **超极化**：K+ 通道关闭较慢，K+以此继续外流导致电位低于静息水平，随后由钠钾泵恢复。
*   **特点 【“全或无”法则 (All or None)】**：
    *   **阈下不反应**：刺激不够不产生 AP。
    *   **强度无关性**：一旦产生，幅度与刺激强度无关（刺激强只增加频率）。
    *   **不衰减性传导**：长距离传输波形大小不变。
    *   **不可叠加性**：存在绝对不应期。

### 4. 局部电位 (Local Potential)
*   **定义**：阈下刺激引起的、不能远传的膜电位波动（如EPSP, IPSP）。
*   **特点**：
    *   **等级性**：幅度随刺激增强而增大。
    *   **衰减性**：随距离迅速衰减（电紧张扩布）。
    *   **可总和**：
        *   **时间总和**：同一部位连续刺激累加。
        *   **空间总和**：不同部位同时刺激累加。

### 5. 局部电位 vs. 动作电位 (对比表)

| 特征 | 局部电位 (Graded Potential) | 动作电位 (Action Potential) |
| :--- | :--- | :--- |
| **振幅** | 随刺激改变 (等级性) | 全或无，恒定 |
| **传播** | 衰减性传导 (局限) | 不衰减传导 (远距离) |
| **总和** | **可以** (时间/空间总和) | **不能** (有绝对不应期) |
| **阈值** | 无 | 有 |
| **机制** | 少量离子通道开放 | 大量电压门控通道顺序开放 |

## C. 突触传递 (Synaptic Transmission)

### 1. 突触分类
*   **按传递方式**：
    *   **化学突触** (Chemical)：利用神经递质，单向，有延迟，主要存在于高等动物。
    *   **电突触** (Electrical)：利用缝隙连接 (Gap junction)，双向，速度快，同步活动。
*   **按连接部位**：轴-树突触、轴-体突触、轴-轴突触。

### 2. 经典化学突触传递过程 (全过程背诵)
1.  **AP到达**：动作电位传导至突触前末梢，导致前膜去极化。
2.  **Ca2+内流**：前膜电压门控 **Ca2+ 通道开放**，Ca2+ 顺浓度梯度内流。
3.  **递质释放**：Ca2+ 触发突触囊泡与前膜融合（胞吐），将神经递质释放到突触间隙。
4.  **受体结合**：递质扩散并结合突触后膜上的特异性受体。
5.  **通道开放**：后膜离子通道开放（或关闭），离子跨膜流动。
6.  **电位产生**：产生突触后电位 (PSP)。
    *   Na+ 内流 -> **兴奋性突触后电位 (EPSP)** -> 去极化。
    *   Cl- 内流或 K+ 外流 -> **抑制性突触后电位 (IPSP)** -> 超极化。

### 3. 电突触 vs. 化学突触 (对比表)

| 特征 | 电突触 | 化学突触 |
| :--- | :--- | :--- |
| **结构基础** | **缝隙连接 (Gap Junction)** | 突触小泡、受体 |
| **间隙距离** | 极小 (2-3 nm) | 较大 (30-50 nm) |
| **传递因子** | 离子流 | 神经递质 |
| **传递方向** | **双向** | **单向** |
| **延迟** | 基本无 (极快) | 明显 (0.3-5 ms) |
| **功能** | 同步性活动 | 信号放大、复杂调制 |

# 第二部分：PPT 课后习题全解 (含详细解析)

本部分包含PPT中出现的所有思考题、主观题及多选题，**直接背诵**即可用于考试。

### 1. 【主观题】关于大脑的 fun fact? (PPT P3)
> **参考答案**：
> *   大脑本身没有痛觉感受器，所以脑外科手术中病人可以是清醒的。
> *   大脑虽然只占体重的 2%，但消耗了人体 20% 的氧气和能量。
> *   人脑中约 75% 是水。
> *   爱因斯坦的大脑重量其实比普通人略轻，但顶叶区域更发达。

### 2. 【主观题】树突棘越多越好吗？(PPT P15)
> **参考答案**：
> **不是越多越好，适量且有效的连接才是关键。**
> *   **发育角度**：在发育早期（婴儿期），树突棘数量确实会爆发式增长（Spinogenesis），这代表了学习潜力的增加。
> *   **修剪机制**：但在青春期，大脑会进行“突触修剪”（Pruning），去除多余、无效的连接，保留强化有效的连接，这是大脑成熟和效率提升的表现。
> *   **病理状态**：
>     *   树突棘**过少**：如阿尔茨海默病，会导致认知功能下降、记忆缺失。
>     *   树突棘**过多或修剪不足**：如自闭症谱系障碍（ASD），可能导致信息处理过载或神经网络连接异常。
> *   **结论**：树突棘的大小、形状和数量处于动态平衡中，关键在于**可塑性**而非单纯的数量。

### 3. 【主观题】树突与轴突的区别？(PPT P20)
> **参考答案**：
> *   **形态结构**：每个神经元通常只有**一根轴突**，细长光滑且直径均匀；而**树突通常有多根**，呈树枝状分支，表面粗糙常有树突棘。
> *   **功能导向**：树突是神经元的**输入端**，负责接收和整合信息（向胞体传导）；轴突是神经元的**输出端**，负责将动作电位传导至末梢（远离胞体）。
> *   **细胞器**：树突内含有核糖体等蛋白合成机构；轴突（特别是轴丘后）基本不含核糖体，不合成蛋白。
> *   **有无髓鞘**：轴突常被髓鞘包裹以加速传导；树突极少有髓鞘。

### 4. 【主观题 - 10分】当脑严重缺氧时会对膜电位有什么影响？为什么？膜电位的外正内负有什么生物学意义？(PPT P52)
> **参考答案**：
>
> **(1) 缺氧对膜电位的影响及原因：**
> *   **影响**：会导致**膜静息电位减小（去极化）**，甚至完全消失，最终导致神经元功能丧失或死亡。
> *   **原因**：
>     1.  **能量耗竭**：维持静息电位（外正内负）主要依赖膜上的**钠-钾泵 (Na+-K+ Pump)**。
>     2.  **ATP依赖**：钠-钾泵是主动运输蛋白，需要消耗 ATP 才能将 Na+ 泵出、K+ 泵入。
>     3.  **缺氧链式反应**：脑缺氧 -> 线粒体有氧呼吸停止 -> ATP 合成中断 -> 钠-钾泵停止工作 -> 膜内外离子浓度梯度无法维持（Na+内流，K+外流） -> 膜电位绝对值下降（去极化）。
>
> **(2) 膜电位“外正内负”的生物学意义：**
> *   **能量储备**：这是一种电化学势能的储备，为神经冲动的产生做好了准备。
> *   **兴奋性基础**：只有维持正常的静息电位，神经元才能对刺激做出反应（产生动作电位）。如果是持续的去极化状态，Na+ 通道会失活，细胞将无法再次兴奋。
> *   **信号传导**：它是神经系统进行编码、传输和处理信息的基础物理状态。

### 5. 【多选题】以下哪项正确地概述了局部电位与动作电位在神经信号传导中的生理机制差异？(PPT P64)
> **题目选项分析**：
> *   A. 局部电位能沿长距离传播而不衰减... (错误，局部电位会衰减)
> *   B. 动作电位在其传播过程中需要达到阈值才能触发，而局部电位的形成不需要达到阈值 (正确)
> *   C. 局部电位的强度与诱发它的刺激强度成线性关系，而动作电位的发放频率与刺激强度有关 (正确)
> *   D. 动作电位可以叠加导致合并... (错误，动作电位有绝对不应期，不可叠加)
>
> **标准答案**： **B, C**

### 6. 【多选题】关于化学突触和电突触的特性，以下哪些是正确的？(PPT P74)
> **题目选项分析**：
> *   A. 化学突触通过直接的离子流... (错误，说反了。电突触是离子流，化学突触是递质)
> *   B. 电突触通常允许信号的双向传递，且传递速度较快 (正确)
> *   C. 化学突触传递信号时不受药物... (错误，极易受药物影响)
> *   D. 电突触在传递信号时不引入突触延迟，而化学突触则可能有延迟 (正确)
> *   E. 化学突触使信号传递具有更高的灵活性和复杂性... (正确)
>
> **标准答案**： **B, D, E**

### 7. 【思考题】如何看待神经胶质细胞在中枢神经系统所扮演的角色？(PPT P76)
> **参考答案**：
> 过去认为胶质细胞只是“胶水”（Glial即胶水之意），现在认为它们是神经系统的**全能管家和合作伙伴**：
> 1.  **结构支持**：构建大脑的支架，分隔神经元。
> 2.  **环境稳态**：星形胶质细胞通过吸收多余的 K+ 和神经递质，维持突触间隙的微环境稳定，防止癫痫等异常放电。
> 3.  **营养与代谢**：连接血管和神经元，输送葡萄糖等营养物质。
> 4.  **防御保护**：形成血脑屏障 (BBB) 阻挡毒素；小胶质细胞作为免疫细胞清除病原体和坏死组织。
> 5.  **信号调节**：绝缘（髓鞘）加速传导；甚至参与突触传递的调节（三突触结构概念），影响可塑性。

### 8. 【思考题】如何理解突触可塑性在学习和记忆中的作用？(PPT P76)
> **参考答案**：
> 突触可塑性（Synaptic Plasticity）是学习和记忆的**神经生物学基础**。
> 1.  **定义**：指突触连接的强度（传递效率）可以随着使用情况而增强或减弱的能力。
> 2.  **长时程增强 (LTP)**：当突触前和突触后神经元同时被激活（Hebbian理论：“Fire together, wire together”），突触连接强度增加。这是**记忆形成**的基础。
> 3.  **长时程抑制 (LTD)**：突触传递效率的降低，用于**遗忘**错误信息或优化神经回路。
> 4.  **结构改变**：学习过程中伴随着树突棘的新生、增大或形状改变（如从细长形变为蘑菇形），增加了受体数量和接触面积，使信号传递更高效。

# 第三部分：高频预测考题库 (补充简答题)

### Q1. 请简述血脑屏障 (BBB) 的组成及其功能意义。
*   **组成**：主要由脑毛细血管内皮细胞（紧密连接）、基膜、以及**星形胶质细胞的血管周足**共同构成。
*   **功能**：
    1.  **物理屏障**：严格限制血液中的物质进入脑组织。
    2.  **保护作用**：阻挡血液中的有害物质、毒素、细菌和病毒进入中枢神经系统。
    3.  **稳态维持**：维持脑内微环境（离子、激素浓度）的相对稳定，保证神经元正常工作。
*   **临床联系**：许多药物无法治疗脑部疾病是因为无法穿过BBB。

### Q2. 为什么说动作电位是“全或无”的？
*   **“全” (All)**：一旦刺激达到阈值，电压门控 Na+ 通道会正反馈式爆发性开放，产生的动作电位幅度和波形是恒定的，达到最大值。
*   **“无” (None)**：如果刺激未达到阈值，Na+ 通道开放数量不足以对抗 K+ 外流，膜电位回到静息水平，不产生动作电位。
*   **意义**：这种特性保证了神经信号在长距离传输中的保真度，信号不会因为传输距离远而衰减消失。

### Q3. 神经纤维髓鞘化的生理意义是什么？(结合少突胶质/施万细胞)
*   **绝缘作用**：髓鞘由脂质构成，具有高电阻、低电容特性，防止电流流失至细胞外液。
*   **跳跃式传导**：动作电位只能在朗飞氏结 (Nodes of Ranvier) 处产生，电流在有髓鞘区段快速流过。这使得神经冲动呈“跳跃式”传导。
*   **结果**：极大地提高了神经冲动的**传导速度**（比无髓纤维快几十倍），同时**节省能量**（只需在节点处进行离子交换）。

### Q4. 比较兴奋性突触后电位 (EPSP) 和抑制性突触后电位 (IPSP) 的产生机制。
*   **EPSP (Excitatory)**：
    *   **递质**：如谷氨酸 (Glutamate)、乙酰胆碱 (ACh)。
    *   **通道**：通透 **Na+** 和 K+ (主要是 Na+ 内流)。
    *   **电位变化**：膜电位**去极化**（向阈电位靠近），使神经元更容易产生动作电位。
*   **IPSP (Inhibitory)**：
    *   **递质**：如 GABA、甘氨酸 (Glycine)。
    *   **通道**：通透 **Cl-** (内流) 或 **K+** (外流)。
    *   **电位变化**：膜电位**超极化**（远离阈电位），使神经元更难兴奋。

### Q5. 简述 Loewi 的双蛙心灌注实验及其结论。
*   **实验设计**：
    1.  刺激第一个蛙心的迷走神经，心率减慢。
    2.  抽取第一个蛙心的灌注液，注入第二个已去掉神经的蛙心中。
    3.  观察到第二个蛙心心率也减慢。
*   **结论**：神经兴奋不是直接通过电流传导给效应器，而是通过释放某种**化学物质**（后来被鉴定为乙酰胆碱 ACh）来传递信号的。证明了**化学突触**的存在。

### Q6. 解释“绝对不应期”和“相对不应期”及其对神经兴奋编码的意义。
*   **绝对不应期**：
    *   **状态**：无论给予多大刺激，神经元都不会再次产生 AP。
    *   **机制**：Na+ 通道处于**失活状态** (Inactivated)，门控无法打开。
    *   **意义**：限制了 AP 的最大发放频率，并迫使 AP 只能单向传导（防止回流）。
*   **相对不应期**：
    *   **状态**：需要比平时更强的刺激才能产生 AP。
    *   **机制**：部分 Na+ 通道复活，但 K+ 通道仍开放（超极化）。
    *   **意义**：允许神经元通过改变 AP 的频率来编码刺激的强度（强刺激可以在相对不应期引发 AP，导致频率增加）。
